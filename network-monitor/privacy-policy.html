プライバシーポリシー

Network Monitor
最終更新日: 2026年2月27日

はじめに

本プライバシーポリシーは、「Network Monitor」（以下「本アプリ」）におけるユーザーの皆さまの情報の取り扱いについて説明するものです。

本アプリは個人開発のアプリケーションであり、ユーザーのプライバシーを最大限に尊重しています。

アプリで使用する情報

本アプリは、macOSのメニューバーに常駐し、リアルタイムのネットワーク通信速度（アップロード・ダウンロード）を表示するアプリです。

本アプリは、以下のデータをアプリの機能提供のために端末上で使用します。これらのデータはすべてユーザーのデバイス上でのみ処理され、外部に送信されることは一切ありません。

  • ネットワーク通信量データ：macOSのシステムAPIから取得するネットワークインターフェースの送受信バイト数。リアルタイムの速度計算にのみ使用され、保存・蓄積されません
  • 設定情報：速度の表示単位などのアプリ内設定。デバイスのローカルストレージにのみ保存されます
✅ 本アプリは、ネットワーク速度の表示のみを目的としており、通信内容の監視や記録は一切行いません。また、氏名・メールアドレス・電話番号・広告識別子などの個人を直接特定する情報は一切使用しません。すべてのデータはデバイス上でのみ処理され、開発者を含む第三者がアクセスすることはできません。

データの保存

本アプリは、クラウドサービス（iCloud等）を一切使用しません。

  • 設定情報はデバイスのローカルストレージにのみ保存されます
  • ネットワーク速度データはメモリ上で一時的に処理され、永続的に保存されることはありません
  • 開発者がユーザーのデータにアクセスする手段は一切ありません

第三者へのデータ提供

本アプリは、ユーザーのデータを第三者に販売、提供、または共有することは一切ありません。そもそもデータを外部に送信する機能自体が存在しません。

広告・トラッキング

本アプリは、広告の表示やユーザーの行動追跡（トラッキング）を一切行いません。

分析ツール、広告SDK、その他のサードパーティのSDKも一切使用していません。

データの削除

本アプリのデータは以下の方法で削除できます。

  • アプリを終了すると、メモリ上のネットワーク速度データは自動的に消去されます
  • アプリを削除すると、すべての設定情報がデバイスから完全に削除されます

お子さまのプライバシー

本アプリは、13歳未満のお子さまから意図的に個人情報を収集することはありません。

免責事項

本アプリの利用は、ユーザーご自身の責任のもとに行われるものとします。本アプリの使用により生じたいかなる損害についても、開発者は一切の責任を負いません。

プライバシーポリシーの変更

本ポリシーは、必要に応じて更新されることがあります。重要な変更がある場合は、本ページにて通知します。

Privacy Policy

Network Monitor
Last updated: February 27, 2026

Introduction

This Privacy Policy describes how the "Network Monitor" application ("the App") handles user information.

This App is developed by an individual developer and respects user privacy to the fullest extent.

Information Used by the App

This App resides in the macOS menu bar and displays real-time network traffic speed (upload and download).

The App uses the following data on your device to provide its features. All data is processed solely on the user's device and is never transmitted externally.

  • Network traffic data: Upload and download byte counts obtained from macOS system APIs. Used only for real-time speed calculation and is not stored or accumulated
  • Settings: In-app preferences such as speed display units. Stored only in the device's local storage
✅ This App is designed solely for displaying network speed and does not monitor or record the content of any network communications. Additionally, no personally identifiable information such as names, email addresses, phone numbers, or advertising identifiers is used. All data is processed only on the device and cannot be accessed by the developer or any third party.

Data Storage

This App does NOT use any cloud services (such as iCloud).

  • Settings are stored only in the device's local storage
  • Network speed data is temporarily processed in memory and is not permanently stored
  • The developer has no means to access any user data

Third-Party Data Sharing

This App does NOT sell, provide, or share user data with any third parties. The App does not even have the functionality to transmit data externally.

Advertising and Tracking

This App does NOT display advertisements or track user behavior in any way.

No analytics tools, advertising SDKs, or any other third-party SDKs are used.

Data Deletion

App data can be deleted in the following ways:

  • When the app is closed, all network speed data in memory is automatically cleared
  • Deleting the app removes all settings data completely from the device

Children's Privacy

This App does not intentionally collect personal information from children under the age of 13.
