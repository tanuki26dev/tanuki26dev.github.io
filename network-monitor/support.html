よくあるご質問

メニューバーに速度が表示されません
システム設定の「プライバシーとセキュリティ」から、Network Monitorに必要な権限が付与されているかご確認ください。また、アプリを再起動してみてください。
表示される速度は正確ですか？
macOSのシステムAPIから取得したネットワークインターフェースの送受信データに基づいて速度を計算しています。実際の通信速度とほぼ一致しますが、VPNやセキュリティソフトの影響で差が生じる場合があります。
通信内容を見られることはありますか？
いいえ。本アプリはネットワークの通信量（バイト数）のみを取得しており、通信内容を監視・記録する機能は一切ありません。
バッテリーへの影響はありますか？
本アプリは軽量に設計されており、バッテリーへの影響は最小限です。1秒ごとにシステムAPIからデータを取得するのみで、大きな処理負荷はかかりません。
利用料金はかかりますか？
本アプリは無料でご利用いただけます。アプリ内課金もありません。

FAQ

The speed is not showing in the menu bar
Please check that Network Monitor has been granted the necessary permissions in System Settings under "Privacy & Security." You may also try restarting the app.
Is the displayed speed accurate?
Speed is calculated based on network interface data obtained from macOS system APIs. It closely matches actual traffic speed, though VPNs or security software may cause slight differences.
Can the app see my network traffic content?
No. This app only reads network traffic volume (byte counts) and has no capability to monitor or record the content of any network communications.
Does it affect battery life?
The app is designed to be lightweight with minimal battery impact. It only retrieves data from system APIs once per second, with negligible processing overhead.
Is there a fee to use the app?
This app is free to use. There are no in-app purchases.

System Requirements

Compatible with Mac running macOS 15.0 (Sequoia) or later.
