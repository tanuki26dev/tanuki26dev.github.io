プライバシーポリシー

買い物リスト - みんなで共有お買い物メモ
最終更新日: 2026年2月25日

はじめに

本プライバシーポリシーは、「買い物リスト」（以下「本アプリ」）におけるユーザーの皆さまの情報の取り扱いについて説明するものです。

本アプリは個人開発のアプリケーションであり、ユーザーのプライバシーを最大限に尊重しています。

アプリで使用する情報

本アプリは、以下のデータをアプリの機能提供のために端末上で使用します。これらのデータはすべてユーザーのiCloudアカウントに保存され、開発者がアクセスすることはできません。

  • 買い物リストデータ：品名、数量、メモ、カテゴリ、購入日などの入力情報
  • プリセットデータ：ユーザーが登録した定型アイテム
  • 位置情報：購入したお店を記録する場合にのみ使用します。常時取得するものではなく、ユーザーがお店を記録する操作を行った時のみ利用されます
  • 設定情報：アプリ内の表示設定や通知設定
✅ 本アプリで入力・記録されたデータは、すべてユーザー自身のiCloudアカウントに保存されます。開発者がこれらのデータを閲覧・取得・利用することは一切できません。また、氏名・メールアドレス・電話番号・広告識別子などの個人を直接特定する情報は一切使用しません。

データの保存と同期

本アプリは、Apple が提供する iCloud（CloudKit） を利用してデータの保存と同期を行います。

  • データはユーザー自身のiCloudアカウントに保存されます
  • 同じApple IDでサインインしたデバイス間でデータが同期されます
  • データの管理はAppleのiCloudサービスに準じます
  • 開発者はユーザーのiCloudデータにアクセスする手段を持っていません

データの共有機能

本アプリには、iCloudの共有機能を利用して他のユーザーと買い物リストを共有する機能があります。

  • 共有はユーザーが明示的に招待した相手にのみ行われます
  • 共有されたデータはAppleのCloudKitインフラストラクチャを通じて安全に転送されます
  • 共有の停止はいつでもユーザー自身が行えます
  • 共有されたデータについても、開発者がアクセスすることはできません

第三者へのデータ提供

本アプリは、ユーザーのデータを第三者に販売、提供、または共有することは一切ありません。開発者自身もユーザーのデータにアクセスすることはできない仕組みになっています。

広告・トラッキング

本アプリは、広告の表示やユーザーの行動追跡（トラッキング）を一切行いません。

分析ツール、広告SDK、その他のサードパーティのSDKも一切使用していません。

データの削除

ユーザーは、アプリ内の操作でいつでもデータを削除できます。

  • 個別のアイテムはスワイプ操作やゴミ箱機能で削除できます
  • アプリを削除するとデバイス上のデータは削除されます
  • iCloud上のデータはAppleの設定から管理できます

お子さまのプライバシー

本アプリは、13歳未満のお子さまから意図的に個人情報を収集することはありません。

免責事項

本アプリの利用は、ユーザーご自身の責任のもとに行われるものとします。本アプリの使用により生じたいかなる損害についても、開発者は一切の責任を負いません。

プライバシーポリシーの変更

本ポリシーは、必要に応じて更新されることがあります。重要な変更がある場合は、本ページにて通知します。

Privacy Policy

Shopping List - Shared Shopping Memo
Last updated: February 25, 2026

Introduction

This Privacy Policy describes how the "Shopping List" application ("the App") handles user information.

This App is developed by an individual developer and respects user privacy to the fullest extent.

Information Used by the App

The App uses the following data on your device to provide its features. All data is stored in the user's own iCloud account, and the developer has no access to it.

  • Shopping list data: Item names, quantities, notes, categories, purchase dates, and other input information
  • Preset data: Pre-registered items saved by the user
  • Location data: Used only when recording the store where an item was purchased. Location is not tracked continuously—it is only accessed when the user chooses to record a store
  • Settings: In-app display and notification preferences
✅ All data entered and recorded in this App is stored in the user's own iCloud account. The developer cannot view, retrieve, or use any of this data. Additionally, no personally identifiable information such as names, email addresses, phone numbers, or advertising identifiers is used.

Data Storage and Sync

The App uses Apple's iCloud (CloudKit) for data storage and synchronization.

  • Data is stored in the user's own iCloud account
  • Data syncs across devices signed in with the same Apple ID
  • Data management follows Apple's iCloud services guidelines
  • The developer has no means to access user iCloud data

Data Sharing Feature

The App includes a feature to share shopping lists with other users through iCloud sharing.

  • Sharing occurs only with people explicitly invited by the user
  • Shared data is securely transferred through Apple's CloudKit infrastructure
  • Users can stop sharing at any time
  • The developer cannot access shared data either

Third-Party Data Sharing

This App does NOT sell, provide, or share user data with any third parties. The developer also cannot access user data due to the app's architecture.

Advertising and Tracking

This App does NOT display advertisements or track user behavior in any way.

No analytics tools, advertising SDKs, or any other third-party SDKs are used.

Data Deletion

Users can delete their data at any time through in-app operations.

  • Individual items can be deleted via swipe actions or the trash feature
  • Deleting the app removes all data from the device
  • iCloud data can be managed through Apple's settings

Children's Privacy

This App does not intentionally collect personal information from children under the age of 13.

Disclaimer

Use of this App is at the user's own risk. The developer shall not be liable for any damages arising from the use of this App.

This policy may be updated as needed. Any significant changes will be communicated on this page.
